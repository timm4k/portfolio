# Portfolio | Tate

Personal portfolio website built with React, TypeScript, and Vite.

## 🚀 Tech Stack

- **Frontend:** React 18+
- **Language:** TypeScript
- **Styling:** CSS3 (Custom architecture)
- **Build Tool:** Vite
- **Deployment:** Vercel

## 📂 Project Structure

- `src/pages` — Main sections (Home, Skills, Projects, About, Contact)
- `src/components` — Reusable UI components
- `src/style` — Global and component-specific styles

## 🛠️ Local Development

1. Clone the repository: `git clone https://github.com/timm4k/portfolio.git`
2. Install dependencies: `npm install`
3. Start dev server: `npm run dev`
